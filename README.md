            Next.js Expense Request (demo frontend)


            Hướng dẫn:
1. Giải nén
2. cd vào thư mục project
3. npm install
4. npm run dev
5. Mở http://localhost:3000/login

Mô tả:
- Trang login demo /expense
- Trang /expense chứa form tạo yêu cầu chi tiêu và danh sách các yêu cầu
- Dữ liệu lưu ở localStorage (khóa 'expense_requests_demo')

Lưu ý: Đây là frontend demo không có backend. Bạn có thể nối API backend để lưu vào DB.
